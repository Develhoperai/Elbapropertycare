# Elba Property Care

A professional static website for **Elba Property Care** – a property maintenance, cleaning, and landscaping service based on the island of Elba, Italy.

## Project Structure

```
├── index.html        # Main HTML page
├── css/
│   └── styles.css    # All styles (responsive, custom properties)
└── js/
    └── main.js       # Interactive behaviour (nav, form validation)
```

## Features

- **Responsive design** – works on desktop, tablet and mobile
- **Sticky header** with active-link highlighting on scroll
- **Mobile navigation** with accessible toggle
- **Services section** – 6 service cards with hover effects
- **About section** with key selling points
- **Testimonials** from satisfied clients
- **Contact / quote form** with client-side validation and success feedback

## Running Locally

Open `index.html` directly in a browser, or serve with any static server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

No build step or dependencies required.
